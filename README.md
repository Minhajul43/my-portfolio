# 🌐 MY Portfolio Website

A modern, responsive, and interactive **Developer Portfolio Website** built using **HTML, CSS, and JavaScript**. This portfolio showcases skills, projects, education, and contact information with a clean UI, smooth animations, and interactive user experience.

Designed to create a professional online presence for developers, students, freelancers, and job seekers.

---

## ✨ Features

✅ Modern & Professional UI Design  
✅ Fully Responsive (Mobile, Tablet & Desktop)  
✅ Smooth Scroll Animations  
✅ Interactive Typing Effect  
✅ Custom Animated Cursor  
✅ Skills Progress Bar Animation  
✅ Filterable Projects Section  
✅ Contact Form UI  
✅ Sticky Navigation Bar  
✅ Scroll Progress Indicator  
✅ Back-To-Top Button  
✅ Hover Effects & Interactive Components
